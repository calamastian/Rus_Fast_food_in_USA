# Поиск оптимального города и района для запуска фастфуд-ресторана на рынке США
*Учебный проект на основе открытого датасета kaggle - Fast Food Restaurants Across America*

Российская сеть ресторанов быстрого питания планирует запуск бизнеса в США и рассматривает несколько приоритетных городов для выхода на рынок.

Задачи:
* оценить привлекательность каждого города с точки зрения юнит-экономики
* определить приоритетные города, количество точек и районы размещения
* подготовить материалы для презентации инвесторам

Условие: 
* 8 городов: Columbus (OH), Cincinnati (OH), Cleveland (OH), Las Vegas (NV), Miami (FL),  Indianapolis (IN), Birmingham (AL), Portland (OR)

Действия:

**Excel**: выполнена первичная обработка внутренних данных компании. Рассчитаны ключевые показатели юнит-экономики: средний чек, структура заказов, себестоимость и операционная прибыль на заказ. Подготовлены сводные таблицы для сравнения показателей по городам.

**Python** (Pandas, NumPy, Seaborn, Folium): проведена очистка, объединение и фильтрация данных по рынку США из нескольких источников. Выполнен разведочный анализ данных (EDA), анализ конкурентной среды и геопространственное моделирование плотности ресторанов. Построены визуализации распределения заведений и интерактивные карты.

**Jupyter Notebook**: задокументирован полный цикл исследования — от подготовки данных до формирования выводов и рекомендаций. Реализованы расчёты показателей конкурентной насыщенности и пространственного распределения заведений.

**Разработка системы оценки городов**: создан комплексный показатель для оценки инвестиционной привлекательности городов, который включает дополнительные демографические и экономические критерии.

Результат: 

Была сформирована единая очищенная база по ресторанам фастфуда в целевых городах США и построена интерактивная карта конкурентной среды. 
Анализ позволил выявить районы с высокой и низкой плотностью заведений, что стало основой для рекомендаций по размещению новых точек.
